# 快速开始指南

## 第一次使用

### 1. 安装扩展

由于这是本地开发的扩展,需要通过调试方式运行:

1. 在VSCode中打开 `big_log_check` 文件夹
2. 按 `F5` 启动扩展开发主机
3. 在新打开的VSCode窗口中使用扩展

### 2. 测试扩展

我们提供了一个示例日志文件 `test_sample.log` 供测试使用。

#### 方式一:通过命令面板打开

1. 按 `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`)
2. 输入 "打开大日志文件"
3. 选择 `test_sample.log` 文件

#### 方式二:通过右键菜单打开

1. 在文件资源管理器中找到 `test_sample.log`
2. 右键点击文件
3. 选择 "打开大日志文件"

### 3. 测试功能

#### 测试搜索功能
- 在搜索框中输入 "ERROR" 或 "INFO"
- 查看高亮显示的搜索结果

#### 测试按时间删除
1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "按时间删除日志"
3. 选择 "删除指定时间之前的日志"
4. 输入时间: `2024-11-19 10:30:00`
5. 确认操作(会删除10:30之前的所有日志)

#### 测试按行数删除
1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "按行数删除日志"
3. 选择 "删除指定行之前的日志"
4. 输入行号: `10`
5. 确认操作(会删除前10行日志)

## 常见问题

### Q: 为什么删除操作后看不到效果?
A: 点击工具栏中的"刷新"按钮,重新加载文件。

### Q: 如何处理自己的日志文件?
A: 直接用相同的方式打开任何 .log 或 .txt 文件即可。

### Q: 支持哪些时间格式?
A: 目前支持以下格式:
- `2024-11-19 10:00:00`
- `2024/11/19 10:00:00`
- `[2024-11-19 10:00:00]`
- `11-19-2024 10:00:00`
- `2024-11-19T10:00:00`

### Q: 如何打包成正式的VSCode扩展?
A: 运行以下命令:
```bash
npm install -g @vscode/vsce
vsce package
```
这会生成一个 `.vsix` 文件,可以通过VSCode的"从VSIX安装"功能安装。
